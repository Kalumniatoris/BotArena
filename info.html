Returns

Bot's AI needs to return specific value to work in array format, where first element is string with action and second one is parameter for that action return ["action", value]; for example return ["LEFT",0.05];

If action have no parameter or if you want to use default one you may use simple format return "action"; for example return "fire";

FIRE

Parameter: none


Attempts to fire bullet directed as bot.

bullets will not harm anybot in the same 'OWNER' group.

Will do nothing if bot's fireCooldown is higher than 0.

LEFT

Parameter: angle in radians. Default value: bot's maximum turn angle


Turns bot left by given angle, if value is higher than bot's maximum angle it would turn by it's maximum value

RIGHT

Parameter: angle in radians. Default value: bot's maximum turn angle


Turns bot right by given angle, if value is higher than bot's maximum angle it would turn by it's maximum value

FASTER

Parameter: none.


Increases speed of bot. Speed will not exceed bot's maxSpeed

SLOWER

Parameter: none.


Decreases speed of bot. Speed may get into negative, which would effect in bot moving backward. Reverse speed will not exceed half of bot's maxSpeed

SPLIT

Parameter: none.


Divides bot into two smaller bots, each bot will have decreased damage and health (both current and maximum) compared to original bot. Divided bots will have the same AI and 'OWNER' group

UPGRADE

Parameter: What to upgrade, options: "health","healing","damage","bulletcount","speed","sight","firerate","bulletspeed".

Upgrades attribute of bot. Requires experience to work. Upgrade costs is calculated: CM * base * (upgradeCount + 1). Where CM is cost multiplier of specific upgrade, base is starting cost (10 on default) and upgradeCount is how many times bot was already upgraded (regardless of upgrade type)


"health" (or "hp"):

Cost multiplier: 1

Increases bot's current and maximum health 1.5 times

"healing":

Cost multiplier: 1

Increases how much bot heals itself using "heal" command. Each upgrade will increase healing efficiency slightly over 2 times.

"damage":

Cost multiplier: 2

Increases strength of bot's bullets 1.5 times

"bulletcount" (or "maxbullets"):

Cost multiplier: 2

Increases how many bullets from bot's 'OWNER' can be on board 1.5 times

"speed":

Cost multiplier: 2

Increases bot's maximum speed 1.2 times and bot's maximum turning angle 1.5 times

"sight"

Cost multiplier: 0.2

Increases area of view of bot. Angle 1.2 times and distance 1.2 times.

"firerate":

Cost multiplier: 2.5

Decreases delay between bullets to 0.8 of current value

"bulletspeed":

Cost multiplier: 1

Increases speed of (new) bullets shoot by bot 1.3 times

HEAL

Parameter: amount. Default value: bot's current experience

Heals bot by amount * bot's healing (1 without upgrades) at cost of amount experience.

Will not overheal.

If specified amount is higher than bot current experience will do nothing.

If healed amount would exceed maximum one excess experience will not be returned.

In short: healing while being completely healthly will take experience without any benefits.


Parameters

Bots have some values accessible to their AI's to read passed as argument to their function, avaliable in read only (except for v) objects

bot

Informations about this bot.


bot.x: Current x position of bot.

bot.y: Current y position of bot.

bot.speed: Current speed of bot.

bot.angle: Current angle of bot.

bot.health: Current health of bot.

bot.maxhealth: Maximum health of bot.

bot.exp or bot.experience: Current experience of bot (gained minus spend on upgrades and healing)

bot.totalExp or bot.totalExperience: Experience gained by bot.

bot.turnSpeed: Maximum angle bot can turn using "LEFT" and "RIGHT" commands.

bot.owner: 'OWNER' value of bot.

bullets

Informations about this bot's bullets.


bullets.count: Amount of bullets of this bot's 'OWNER' on arena.

bullets.max: Maximum amount of bullets on arena allowed for this bot's 'OWNER'

bullets.speed: Speed of bullets shoot by this bot

bullets.damage: Damage that bullets shoot by this bot could deal

arena

Size of arena.


arena.width: Width of current arena.

arena.height: Height of current arena

seen

Array of other bots 'detected' by this bot.


seen[id].distance: Distance to that bot.

seen[id].angleTo: Angle relative to that bot.

seen[id].owner: 'OWNER' of that bot.

seen[id].health: Current health of that bot.

costs

Current costs of upgrades avaliable to this bot.


cost.(health/healing/firerate/speed/bulletspeed/sight/damage/bulletcount): Cost for next upgrade for that attribute

v

Object avaliable to user, doesn't contain any values by default, values here are saved between bots with the same 'OWNER' (and between steps).

Example usage


v.test=10: saves 10 into variable v.test

var test=v.test: reads (semi-global) variable v.test into local variable test
